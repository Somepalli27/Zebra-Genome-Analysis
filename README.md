# Zebra Genome Sequence Analysis

This project analyzes a zebra genomic DNA sequence using Python and Biopython.

Features:
- Read FASTA file
- Calculate sequence length
- Count A=1149, T=1273, G=1680, C=1653
- Calculate GC content
- Generate reverse complement
- Translate DNA to protein

Results:
Length: 5755 bp
GC Content: 57.91%
